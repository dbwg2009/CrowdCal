# Migrations

Place Drizzle ORM migration files here. Use Drizzle Kit to generate and run migrations for the D1 database.

Example commands:

```
npx drizzle-kit generate:sqlite --schema=./schema.ts --out=./migrations
npx drizzle-kit push:sqlite --schema=./schema.ts --out=./migrations
```

See Drizzle ORM docs for more details.
